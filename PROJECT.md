## 1. Обзор продукта

Название (рабочее): Personal Finance Tracker
Тип: Финтех-приложение (web/mobile)
Цель: Помочь пользователям отслеживать, анализировать и контролировать личные и семейные расходы.

Система предназначена для:

учета доходов и расходов
контроля денежного потока (cash flow)
категоризации затрат
работы с несколькими валютами
ведения совместного семейного бюджета
учета периодических доходов

## 2. Основные функции (Core Features)
### 2.1 Учет расходов
Добавление записи о расходе:
сумма
валюта
дата покупки
категория
подкатегория
комментарий (опционально)
Редактирование и удаление записей
Просмотр списка расходов (с фильтрацией и сортировкой)

### 2.2 Периодические доходы (Recurring Income)

Поддержка регулярных поступлений:

Параметры:

сумма
валюта
категория
дата начала
частота:
    ежедневно
    еженедельно
    ежемесячно
    ежегодно
(опционально) дата окончания

Поведение:

система автоматически создает транзакции
или
отображает ожидаемые доходы (в зависимости от реализации)

### 2.3 Категории и подкатегории
Системные категории (по умолчанию)
Пользовательские категории:
создание
редактирование
удаление
Подкатегории:
привязка к категории
кастомизация пользователем

Пример:

Еда
Рестораны
Продукты
Транспорт
Такси
Общественный транспорт

### 2.4 Поддержка валют
Список валют (преднастроенный + кастомный)
Возможность:
добавлять валюты
удалять/скрывать валюты
Хранение расходов в исходной валюте
(Опционально, future scope) конвертация в базовую валюту

### 2.5 Многопользовательская система
Поддержка нескольких пользователей в одном пространстве (например, семья)
Возможности:
добавление участников
разграничение прав (см. роли)
Отображение автора записи

### 2.6 Фильтрация и поиск
Фильтры:
по дате (диапазон)
по пользователю
по категории/подкатегории
по валюте
Поиск по тексту (комментарии)

### 2.7 Отчеты и аналитика (MVP+)
Сумма расходов:
за период
по категориям
Простые визуализации (диаграммы)
Разбивка по пользователям

## 3. Роли пользователей
### 3.1 Владелец (Owner)
Создает пространство (аккаунт/семью)
Управляет:
пользователями
валютами
категориями
Полный доступ ко всем данным
### 3.2 Участник (Member)
Может:
добавлять/редактировать свои расходы
просматривать все расходы (если не ограничено)
Ограниченные права на настройки (по решению владельца)
### 3.3 (Опционально) Наблюдатель (Viewer)
Только просмотр
Без права редактирования
